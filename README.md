# Threat-Detection-Simulation-Wazuh-SIEM
# Purpose
- Deploy and configure Wazuh SIEM using an all-in-one setup.
- Collect logs from multiple sources, including Windows Event Log, Sysmon, and Linux Syslog/Auditd.
- Build custom decoders and detection rules mapped to MITRE ATT&CK techniques.
- Simulate real-world attack scenarios using Atomic Red Team across multiple tactics.
- Evaluate detection rate and visualize security events on the Kibana dashboard.
# Technologies Used:
Wazuh 4.x · Elastic Stack · Sysmon · Atomic Red Team · MITRE ATT&CK Framework
